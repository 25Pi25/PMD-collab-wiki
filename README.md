# PMD-collab-wiki

PMD Collab Site
Each assets comes from PMD Collab github. Can be change be the `CDN_URL` parameter in src/types/enum.ts.

## Installation

```
npm install
```

## Dev

```
npm start
```

## Deploy

it creates a gh-pages branch in your repo
```
npm run deploy
```

## Update type tracker definition

```
npm run deploy
```
